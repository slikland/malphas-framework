Styles

There are some set of css styles that are pre-defined.

Colors

There are 5 predefined colors that can be used as CSS classes.
The set of colors are:
#d9534f: Red
#f0ad4e: Orange
#5bc0de: Light blue
#337ab7: Dark blue
#5cb85c: Green

It uses indexes from 1 to 5 with the following prefixes.

bg-: sets the background-color to the predefined color
text-: sets the color to predefined color
button-: Sets background-color to predefined color and the color to a contrasting color (either a dark gray or white)
tag-: Sets background-color to a 10% lighter color than the predefined, border-color to 10% darker, and color to a contrasting color (either a dark gray or white)
<button class="button-1">Red button</button>

There are also dynamic colors that are used as CMS Theme color set in the Setting.
When the cms/index.php is loading, the PHP will replace all _color_ in main.css to the predefined colors set in the database in Setting table.
