Behavior: Draggable

Selector: '[draggable]'

The draggable attribute indicates that the HTMLElement can be dragged.
The attribute should be a unique name to the draggable elements.

Drop area

To tell where the draggable element can be dropped, the HTMLElement that accepts the drop need to have the attribute draggableContext equal to the draggable attribute.

<div draggableContext="dragger1">
    <div draggable="dragger1">Drag me</div>
    <div draggable="dragger1">Drag me</div>
</div>
<div draggableContext="dragger1">
<!-- Another drop area to the above elements -->
</div>
