Validation

The validation component is a markup to validate the input data.

All types of validation is a markup so it's easy to manipulate the data needed for validation.

<field>
	<label>E-mail</label>
	<input type="text" name="email" focus="1" />
	<validation>
		<email>
			<message>Invalid e-mail format.</message>
		</email>
		<required message="This field is required."></required>
	</validation>
</field>

Validations

For now, you can only use predefined validations which are:

email: Email validation

required: Verify if the field is empty

match: Match the field with another
<field>
	<label>Password:</label>
	<input name="password" />
</field>
<field>
	<label>Confirm password:</label>
	<input name="confirm_password" />
	<validation>
		<match field="password"></match>
	</validation>
</field>

regex: Test with regex pattern
Remember that you'll need to use double backslash to escape the escape character.
<field>
	<label>Some field:</label>
	<input name="test" />
	<validation>
		<regex pattern="/^[\\d]+$/"></regex>
	</validation>
</field>
