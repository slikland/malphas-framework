Directory Structure

The CMS Framework has 2 main directories.

/api - Back-end services
/cms - CMS Front-end

Back-end

The Back-end system is a independent system which can be used without the CMS Front-end.
If the system does not need a CMS system, only some Back-end, this Back-end system can be used as a Standalone

Front-end

The Front-end system depends on the /api Back-end system.
It's written in CoffeeScript and Stylus and uses few 3rd party system.
