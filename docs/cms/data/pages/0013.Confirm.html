Behavior: Confirm

Selector: '[confirm]'

When this attribute is set, the browser's window.confirm method is invoked with the message set in confirm attribute.
If the user selects "OK" in the confirm dialog, the flow continues.
If user selects "Cancel" it'll prevent and stop propagation of click event, therefore will cancel the click.

<button action="../api/cms/user/remove" confirm="Are you sure?">REMOVE USER</button>
