Toggle

The toggle component creates a element that toggles between two different states.

Inside the Toggle markup, should contain default and selected markups which are the states of toggle.
If inside the Toggle markup a input element is not found, it creates automatically as a checkbox to store the current state and have a easy access to Form components.

<toggle>
Something is 
<default>not selected</default>
<selected>SELECTED!</selected>
</toggle>
This way you can have any kind of toggle buttons, with any kind of content inside, if a icon, button, or just a text.
