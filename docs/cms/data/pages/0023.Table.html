Table

The table also doesn't have a specific class only some css styles.
You should use the correct table HTML markups in order to have the correct behaviors.
<table>
	<thead>
		<tr>
			<th>Table header1</th>
			<th>Table header2</th>
		</tr>
	</thead>
	<tbody>
		<tr>
			<td>Row1 Col1</td>
			<td>Row1 Col2</td>
		</tr>
		<tr>
			<td>Row2 Col1</td>
			<td>Row2 Col2</td>
		</tr>
    </tbody>
</table>
Service may be used to create the table.
Also, sort attribute should be used to sort the columns dynamically.

Here's an example of a full table with sort, search and filters also pagination.
card.col-full: 
	header: "Activity log"
	content:
		field: 
			label: "Search"
			input: {"for":"log_table", "name":"search"}
		field: 
			label: "Filter"
			group: {"service":"../api/cms/user/getRoles", "noloading":1}
				field:
					input: {"for":"log_table", "name":"filter_fk_cms_role", "type":"checkbox", "value":"#{id}"}
					label: "#{name}"
		table: {"id":"log_table", "service": "../api/cms/user/getLog"}
			thead:
				tr:
					th: {"sort":"name", "html":"Usuário", "width": "1%"}
					th: {"sort":"action", "html":"Ação"}
					th: {"sort":"description","html":"Descrição"}
					th: "Dados"
					th: {"sort":"created","html":"Data","width":"1%"}
			tbody.text-small:
				>logRow: #{items}
		pagination: {"for":"log_table", "numItems": "20"}
<logRow:
	tr:
		td.nowrap: {"html":"#{name}"}
		td: {"html":"#{action}"}
		td: {"html":"#{description}"}
		td: {"html":""}
		td.text-right.nowrap: {"html":"#{created}"}
In this example, the Service behavior will place all parameters in the URL as URL Vars according to the id attribute, in this case log_table, so the parameters that build each component would be stored in the URL.
This way the page can be reloaded or shared without losing data.
The sort attribute, when the data is loaded, is sorted according to it's name, and if there's a minus sign (-) before the name, it inverts the sorting.

Caveat

When sorting, holding the CMD key and clicking, will concatenate the sorting in order that was clicked.
When clicked without holding CMD key it'll reset the sorting.
