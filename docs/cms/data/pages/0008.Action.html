Behavior: Action

Selector: ':not(form)[action]'

The Action behavior is used to send data to the back-end.
It makes a POST request to the URL, without any data. It works similar to the action attribute of HTML Form element.

Success attribute

It can have a success attribute which indicates what it does when the request is complete.
Success attribute can be one of the below:

refresh: Will re-render the current page
reload: Will reload the entire window (Browser refresh)
URI of CMS page: will change the CMS page.
