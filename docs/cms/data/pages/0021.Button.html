Button

There's no specific Class for the Button, but a button should usually have a Href or Action attribute.
If it works as a submit button in a Form component, you should not set either Href or Action.

<button action="../api/cms/just/call/some/service" confirm="Are you sure?">Click me</button>
<button href="another/page">Goto another page</button>
<form action="../api/cms/save/something">
    <button>Submit</button>
</form>
